📖 Как пользоваться программой ›
1
Получи YouTube API ключ

Нужен для поиска видео и загрузки каналов. Бесплатно, квота 10 000 запросов/день (~100–200 поисков).

  1. Зайди на console.cloud.google.com
  2. Создай проект → APIs & Services → Enable APIs → YouTube Data API v3
  3. Credentials → Create Credentials → API Key
  4. Скопируй ключ и вставь в поле выше
2
Найди видео

Два способа:

  • По каналу — вставь @handle или ссылку на канал, укажи сколько последних видео загрузить
  • По запросу — введи тему (например «машинное обучение»), выбери сортировку и период
3
Выбери видео и запусти транскрибацию

Отметь нужные видео галочкой и нажми ▶ Транскрибировать. Прогресс показывается в реальном времени.

Оценка времени выполнения появляется автоматически после загрузки списка.

4
Как работает транскрибация
  • 🏃 Субтитры YouTube — если у видео есть субтитры, берёт их мгновенно (~3 сек)
  • 🎙 Whisper AI — если субтитров нет, скачивает аудио и транскрибирует локально на твоём компьютере. На Apple M-серии работает быстро (~2–5 мин/час видео).
5
Найди и экспортируй результаты
  • Библиотека — все транскрипции с поиском по тексту
  • Экспорт — выгрузить в Obsidian vault или скачать ZIP-архивом
  • Настройки — выбери папку для сохранения, режим нагрузки и формат файлов

Режим сохранения

Каждое видео в свой файл — или все транскрипции в один общий файл с разделителями.

Временные метки

Добавлять [ЧЧ:ММ:СС] к каждой фразе. Удобно для навигации по тексту, особенно для длинных видео.

Режим нагрузки

Сколько видео обрабатывается параллельно. Больше потоков — быстрее, но сильнее грузит процессор.
